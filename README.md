# Viagens — Landing Page

Uma **landing page simples e responsiva** criada como projeto de prática em HTML e CSS, com o tema de **vilarejos europeus e turismo**.  
O objetivo é apresentar uma página de viagens que desperte o interesse do visitante em explorar novos destinos.

## Sobre o Projeto

Esta landing page foi desenvolvida para praticar os conceitos de:

- Estrutura semântica do **HTML5**  
- Uso de **CSS3** para estilização e responsividade  
- Criação de **navegação interna** entre seções com `id` e `href`  
- Utilização de **efeitos de hover e transições suaves**  
- Aplicação de **boas práticas de acessibilidade** com `alt` nas imagens
